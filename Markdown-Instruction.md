## Работа с изображениями

Чтобы вставить изображение в текст, используй следующую комадну:

![ Описание изображения ] ( путь к изображению )

Заменяющий текст необходим для средств чтения с экрана для слабовидящих. Он также удобен при возникновении ошибок на сайте, из-за которых не удается воспроизвести изображение.
