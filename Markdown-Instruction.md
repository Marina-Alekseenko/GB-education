# Инструкция для работы с MarkDown

## Выделение текста

*Чтобы использовать курсив, введи знак * в начале и конце строки без пробела*

**Чтобы использовать полужирный шрифт, введи знак ** в начале и конце строки без пробелов**

~~Чтобы использовать зачеркнутый шрифт, введи знак ~~ в начале и конце строки без пробела~~

## Списки

* для создания ненумерованного списка, в начале строки введи знак *

1. для создания нумерованного списка в начале строки введи порядковый номер строки списка с точкой - 1. или 2. или 3.
